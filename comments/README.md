# Comments

Reactions to external things — a blog post, a video, a paper, a post.

Keep a copy here whether or not it's also posted elsewhere (own your words). Anchored to a target, dated, not rewritten.

## Conventions

- Light reactions → a monthly file (`{yyyy}-{mm}.md`), one entry per item.
- Heavier responses → a file of their own.
- Link to what you're reacting to. A comment that grows into a standalone piece graduates to `notes/`.
